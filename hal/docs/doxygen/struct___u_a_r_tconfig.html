◆ mode

unsigned int _UARTconfig::mode

Value to be written to the UART mode register. For UART0, use only normal (AT91C_US_USMODE_NORMAL) mode for the lowest 4 bits.
For UART2, use only amongst:
normal (AT91C_US_USMODE_NORMAL) or
hardware-handshaking (AT91C_US_USMODE_HWHSH)
Use only MCK for Clock Selection. See AT91SAM9G20 datasheet.

◆ rxtimeout

unsigned short _UARTconfig::rxtimeout

Timeout value for reception in baudrate ticks, so timeout in seconds is value / baudrate. The timeout only starts counting after the first byte of the transfer has been received. If a timeout is specified it affects all read functions (UART_read, UART_writeRead, UART_queueTransfer).

◆ timeGuard

unsigned char _UARTconfig::timeGuard

Value to be written to the timeguard register. An idle time of (timeGuard*bitPeriod) is added after each byte. See AT91SAM9G20 datasheet.
